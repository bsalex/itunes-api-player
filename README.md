# ITunes API Player
The app supports only *Google Chrome >= 69* (it has not been tested against anything else).  

### Technologies used:  
  - React 16
  - recompose
  - redux
  - redux-saga
  - create-react-app

### Supported features:
  - Search
  - Mobile (width <= 700px) and desktop versions
  - Playback, pause, stop

### How to build:

  - Install node.js 10.x

```
cd {PROJECT_ROOT}
npm install
npm run build
```

### How to run development version:
```
cd {PROJECT_ROOT}
npm start
```
